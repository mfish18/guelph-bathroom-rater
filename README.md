🚽 University of Guelph Bathroom Rater
A community-driven web application that helps University of Guelph students find and rate the best bathrooms on campus. Built with Next.js, TypeScript, and Firebase.
https://guelph-bathroom-rater.vercel.app/

🎯 Features

Interactive Campus Map - Browse bathroom locations across campus
Rate Bathrooms - Rate on cleanliness, supplies, and smell (1-5 stars)
Average Ratings - See community averages for each bathroom
Admin Controls - Only authorized users can add/delete bathrooms
No Spam - Users can only rate each bathroom once per session
Real-time Sync - All ratings sync instantly via Firebase
Fully Responsive - Works on desktop, tablet, and mobile

🛠️ Tech Stack

Framework: Next.js 
Language: TypeScript
Database: Firebase Firestore
Authentication: Firebase Anonymous Auth
Maps: React-Leaflet + OpenStreetMap
Styling: Tailwind CSS
Icons: Lucide React
Hosting: Vercel
